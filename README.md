# Сжатие цветных изображений с использованием сверточного автоэнкодера

Репозиторий содержит проект по исследованию и применению глубокого обучения (нейросети типа автоэнкодер) для уменьшения размера (сжатия) графических файлов в трехканальном цветовом пространстве с минимизацией потерь визуального качества.

## 🚀 Основные особенности проекта
* **Архитектура**: Симметричная многоуровневая зеркальная структура (Энкодер -> Скрытый код -> Декодер).
* **Кастомная функция потерь (Total Loss)**: Комбинация трех метрик оценки качества изображений — MS-SSIM, Chroma Loss и MSE.
* **Производительность**: Модель эффективно сжимает графические файлы высокого разрешения, превосходя по коэффициенту сжатия классические алгоритмы (RLE, Huffman, LZW) на сложных выборках.

---

## 📐 Архитектура нейронной сети

В качестве математической основы используется сверточный автоэнкодер. Система объединяет два связанных модуля:
1. **Сжимающий блок (Энкодер)** — преобразует исходную информацию высокого разрешения в сокращенный скрытый код (латентное представление).
2. **Восстанавливающий блок (Декодер)** — генерирует полноразмерное изображение из латентного кода.

### Структура энкодера
Исходное изображение имеет разрешение `300x300` пикселей в трехканальном цветовом пространстве (RGB).
* **Сверточный слой**: Использует 16 обработчиков (ядер) размером `3x3` с шагом (stride) = 1 и симметричным расширением границ (padding='same'), что сохраняет геометрические пропорции.
* **Функция активации**: Поэлементный ReLU, обнуляющий отрицательные значения:
  $$f(x) = \max(0, x)$$
* **Субдискретизация (Max Pooling)**: Окно `2x2`, шаг = 2. Линейные размеры изображения сокращаются в 2 раза на каждом этапе.

## 📉 Процесс обучения и функция потерь

Модель обучается на выборке из **1000 изображений**. Для достижения наилучшего визуального сходства оптимизируется комплексный показатель **Total Loss**, состоящий из:
* **MS-SSIM (Multi-Scale Structural Similarity)** — оценивает сохранение структурных компонентов, текстур и контраста.
* **Chroma Loss** — отвечает за корректность цветопередачи и минимизацию искажений палитры.
* **MSE (Mean Squared Error)** — среднеквадратичная ошибка, оптимизирующая попиксельное совпадение.

### Динамика обучения (300 эпох):
В ходе обучения зафиксированы следующие показатели сходимости:
* **Эпоха 1**: Total Loss: `52.02` | MS-SSIM: `0.4143` | Chroma Loss: `155.63` | MSE: `0.0132`
* **Эпоха 50**: Total Loss: `9.54` | MS-SSIM: `0.1800` | Chroma Loss: `28.46` | MSE: `0.0032`
* **Финальные показатели**: Значение MS-SSIM опустилось до уровня `0.12`, Chroma Loss — до `10` условных единиц, а MSE составило `0.0023`. Чем ближе данные показатели к 0, тем точнее восстановление.
